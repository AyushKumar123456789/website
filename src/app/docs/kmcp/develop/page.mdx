# Develop MCP servers

Use kmcp to quickly create and initialize an MCP project on your local machine. kmcp automatically generates all the necessary files and dependencies so that you can get started with developing your MCP server and tools. 

Select one of the following MCP frameworks to get started: 

* [FastMCP Python](/docs/kmcp/develop/fastmcp-python)
* [MCP Go](/docs/kmcp/develop/mcp-go)
