# Develop MCP servers

Use KMCP to quickly spin up your MCP project on your local machine. KMCP automatically initializes your project with all the necessary files and dependencies so that you can get started with developing your MCP server and tools. 

KMCP supports the following MCP frameworks: 

* [FastMCP Python](/docs/kmcp/develop/fastmcp-python)
* [MCP Go](/docs/kmcp/develop/mcp-go)
