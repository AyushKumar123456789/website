# Deploy the controller

1. Create a personal access token in GitHub or use an existing one. You can follow the instructions in [???]() to create your token. Make sure that the token has the `read: package` permission. 
2. Encode your GitHub username and personal access token to base64. 
   ```sh
   echo -n "USERNAME:PERSONAL_ACCESS_TOKEN" | base64
   ```
3. Create a `config.json` file with the following content. 
   ```sh
   cat << EOF > data.json
   {
      "auths": {
         "ghcr.io": {
         "auth": "[BASE64-ENCODED-USERNAME:TOKEN]" 
         }
      }
   }
   EOF
   ```
4. Deploy the KMCP controller. 
   ```sh
   kmcp deploy controller --registry-config config.json --version v0.0.1-test2
   ```
