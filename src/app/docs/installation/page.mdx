# Installing Kagent

This guide covers ways to install and configure kagent in your Kubernetes environment. For a quick setup, check out our [Quick Start Guide](/docs/quickstart).

## Installation Methods

### Using Helm (Recommended)

The recommended way to install kagent is using Helm - make sure you set the `OPENAI_API_KEY` environment variable:

```bash
helm install kagent oci://ghcr.io/kagent-dev/kagent/helm/kagent \
    --namespace kagent \
    --create-namespace \
    --set openai.apiKey=$OPENAI_API_KEY \

```

## Uninstallation

To remove kagent from your cluster:

```bash
helm uninstall kagent
```
