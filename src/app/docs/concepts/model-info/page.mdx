# Model Information

The ModelInfo configuration specifies or overrides a model's capabilities in kagent. It's required for custom models and optional for pre-defined models.

## Example

```yaml
apiVersion: kagent.dev/v1alpha1
kind: ModelConfig
metadata:
  name: model-config
  namespace: kagent
spec:
  model: my-model
  provider: CustomProvider
  modelInfo:
    vision: false
    functionCalling: true
    jsonOutput: true
    family: "custom"
    structuredOutput: true
    multipleSystemMessages: false
```
